```python
def fingerprint_batch_embed_image(
    metadata: dict[str, Any], ordered_keys: list[str]
) -> str:
    """SHA-256 hex of canonical JSON for batch_embed_image inputs and ordered key list."""
    min_rating_raw = metadata.get("min_rating")
    min_rating: int | None
    try:
        min_rating = int(min_rating_raw) if min_rating_raw is not None else None
    except (TypeError, ValueError):
        min_rating = None
    pairs = sorted(ordered_keys)
    payload = {
        "date_filter": metadata.get("date_filter", "all"),
        "embedding_dim": CLIP_EMBED_DIM,
        "embedding_model_id": CLIP_EMBED_MODEL_ID,
        "force": bool(metadata.get("force", False)),
        "image_type": str(metadata.get("image_type", "catalog")),
        "min_rating": min_rating,
        "pairs": pairs,
    }
    canonical = json.dumps(payload, sort_keys=True, separators=(",", ":"))
    return hashlib.sha256(canonical.encode("utf-8")).hexdigest()
```

Keep payload keys aligned with `fingerprint_batch_text_embed` (same metadata keys); only `embedding_dim`, `embedding_model_id`, and `pairs` content differ.

```python
def handle_batch_embed_image(runner, job_id: str, metadata: dict) -> None:
    """Embed catalog images into ``image_clip_embeddings`` (sqlite-vec)."""
    with cancel_scope.install(lambda: runner.is_cancelled(job_id)):
        _handle_batch_embed_image_inner(runner, job_id, metadata)
```

4. Implement `_handle_batch_embed_image_inner` by copying the control flow of `_handle_batch_text_embed_inner` with these substitutions:

- If `metadata.get('image_type', 'catalog') != 'catalog'`: `runner.fail_job(job_id, 'batch_embed_image only supports catalog images', severity='warning')` and `return`.
- After `init_database`: log `batch_embed_image: model={CLIP_EMBED_MODEL_ID}` via `add_job_log`.
- `months, year = _resolve_date_window(metadata)`; parse `min_rating` like text embed; `force = bool(metadata.get('force', False))`.
- `full_list: list[str] = list_catalog_keys_for_clip_embed_force(lib_db, months=months, year=year, min_rating=min_rating)` if `force` else `list_catalog_keys_needing_clip_embedding(...)`.
- `fp = fingerprint_batch_embed_image(metadata, full_list)`.
- `processed_pairs: set[str] = set()`; load checkpoint when `chk.get('job_type') == 'batch_embed_image'` and `chk.get('checkpoint_version') == 1` and `chk.get('fingerprint') == fp`; if fingerprint differs and chk had one, log `checkpoint mismatch: batch_embed_image fingerprint changed, starting fresh` (same wording pattern as text embed).
- `remaining = [k for k in full_list if k not in processed_pairs]`.
- Progress: 5% after discovery; 5–95% linear on `embedded / total_at_start`; `clear_checkpoint` on zero work and on success.
- `persist_checkpoint` payload: `{'job_type': 'batch_embed_image', 'fingerprint': fp, 'processed_pairs': sorted(processed_pairs), 'total_at_start': total_at_start}`.
- Main loop: for each key in `remaining`, check cancel; fetch filepath with `lib_db.execute("SELECT filepath FROM images WHERE key = ?", (key,)).fetchone()`; if no row or empty filepath or not `os.path.isfile(filepath)`: `skipped += 1`, warning log, add `key` to `processed_pairs`, `persist_progress`, continue.
- Batch buffer: lists `buf_keys: list[str]`, `buf_paths: list[str]`; flush when `len(buf_keys) >= 8` (constant `_BATCH_EMBED_IMAGE_SIZE = 8` at module level next to other handler constants if any).
- `flush_batch`: `vecs = encode_images(buf_paths, batch_size=8)`; for each index `j`, `numpy_to_clip_vec_blob(vecs[j])` then `with library_write(lib_db): upsert_image_clip_embedding(lib_db, buf_keys[j], vec_blob)`; `processed_pairs.add(buf_keys[j])`, `embedded += 1`, progress update, `persist_progress`.
- `complete_job` with `{'embedded': embedded, 'skipped': skipped, 'failed': failed, 'total': total_at_start}` (`failed` may stay 0 unless you add per-item try/except).
- `except`/`finally`: mirror text embed (`fail_job`, close `lib_db`).

5. Register in the `JOB_HANDLERS` dict: `'batch_embed_image': handle_batch_embed_image`.
