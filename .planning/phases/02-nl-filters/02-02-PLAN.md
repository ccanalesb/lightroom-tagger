## Objective

Add **`POST /api/images/nl-search`** (D-01) that always invokes the LLM via **`ProviderRegistry` + `FallbackDispatcher.call_with_fallback` + `vision_client.complete_chat_text`** (D-04, D-05, RESEARCH), validates output as **`CatalogNlFilter`**, runs **`query_catalog_images`** with the derived kwargs, and returns **JSON** **`{"filters": <object>, "total": <int>, "images": <array>}`** matching **`GET /api/images/catalog`** row shape (D-02, D-03). Map Pydantic / **`ValueError`** from DB to **HTTP 400** (D-09, D-10). Add API tests with **mocked LLM** and **`monkeypatch` `utils.db.LIBRARY_DB`**.

**Depends on:** plan **02-01** (model + `query_catalog_images` array filters). The JSON key for the list of catalog rows is **`images`** (canonical, aligned with `GET /api/images/catalog`); do not use `results` for that list.
