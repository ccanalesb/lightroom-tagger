<threat_model>
**ASVS level:** 1  
**Block on severity:** high  

**Threats and mitigations:**
- **T-19-XSS (captions / reasoning):** Escape all user/model text with **`html.escape(..., quote=True)`** before concatenating into HTML.
- **T-19-path disclosure:** Primary **`main`** contains only **`media_key`**, relative asset URLs, and scores — never **`debug_resolved_path`** or Instagram **`file_path`**.
- **T-19-path traversal in asset names:** Asset basenames must match **`^[a-zA-Z0-9][a-zA-Z0-9._-]*$`** (no **`..`**, no **`/`**); use a deterministic safe stem such as **`ig_{sha256(insta_key)[:12]}_r{rank}_{sha256(catalog_key)[:8]}.jpg`** (concrete formula in task).
- **T-19-oversized outputs:** Honor **`--limit`** when provided; without **`--limit`**, document in module docstring that large catalogs may produce large folders (operator responsibility).

**Residual:** `file://` origin — no CSP; escaping mitigates injection.
</threat_model>
