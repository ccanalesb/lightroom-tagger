## Tool architecture

**3 tools exposed to LLM:**
- `search_catalog(description_search, score_perspective, sort_by_score, sort_by_date, min_score, min_rating, limit)` — main workhorse; model sets limit=1 for "the best"
- `get_scoring_perspectives()` — runtime discovery of valid slugs before searching by score
- `filter_by_date(date_from, date_to, sort_direction, limit)` — temporal queries

**Tool result payload (per image):** key, filename, date_taken, score, score_perspective, score_rationale, description, mood_tags, dominant_colors
