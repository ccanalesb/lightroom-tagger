<threat_model>
**ASVS L1 — read-only insights**

- **Threat:** Expanded attack surface from new route — mitigate with same **`@with_db`** and error handling as `analytics.py`; no raw query params passed into SQL strings.
- **Privacy:** Overview JSON may bundle fingerprint + best photos; treat like existing identity responses (local trusted deployment).
</threat_model>
