<objective>
Replace the legacy **Instagram-match-centric** home (`DashboardPage`) with an **Insights** experience that **composes** existing APIs (**09-01** optional overview endpoint is **not** required per **D-52**): **KPI row** from `SystemAPI.stats` (+ optional active job count), **aggregate score distribution** from fingerprint `aggregate_distribution`, **per-perspective radar** from `per_perspective`, **mini posting frequency** from `AnalyticsAPI.getPostingFrequency` with a **fixed default range**, **top-scored photos** from `IdentityAPI.getBestPhotos`, and **quick-nav cards** to `/analytics`, `/identity`, and `/processing`. Implement **loading**, **per-section errors** (`Promise.allSettled` pattern like `AnalyticsPage`), and **empty states** using identity `meta.coverage_note` / `empty_state` patterns and analytics zero-post hints.
</objective>
