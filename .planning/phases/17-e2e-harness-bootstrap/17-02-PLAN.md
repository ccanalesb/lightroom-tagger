Implementation rules (verbatim behavior contracts):

1. **Transport:** **`subprocess.run`**, invoking the **`browser-harness`** executable **`[Path(shutil.which("browser-harness") or "").as_posix()]`** guarded: if **`which`** returns **`None`**, raise **`RuntimeError`** whose message contains substring **`browser-harness not found on PATH`** and instruct **`browser-harness --doctor`** (exact substring **`browser-harness --doctor`** in the message).

2. **`navigate(url)` stdin script:** two lines **`new_tab("URL")`** (URL substituted, JSON-escaped quoting where needed via Python **`repr()`** injected into snippet) followed by **`wait_for_load()`**.

3. **`get_text(css_selector)` stdin script:** **`js`** call must `return document.querySelector(SELECTOR)?.innerText || ""` — choose safe quoting: **`json.dumps(css_selector)`** embedded into **`js`** argument string generated in Python (no brittle manual escaping).

4. **`wait_for(css_selector, timeout_seconds=...)`:** implement polling in Python (loop with **`time.monotonic`**), each iteration running a **`browser-harness`** snippet returning boolean via **`js`** equivalent of **`document.querySelector(SEL) !== null`**. Raise **`TimeoutError`** with message substring **`timeout waiting for`** and include the **`css_selector`** string if unmatched after **`timeout_seconds`**.

5. **`click(css_selector)`:** stdin script retrieves element center via **`js`** (**`getBoundingClientRect`** on **`document.querySelector`**) computing **`int(centerX)`**, **`int(centerY)`** then calls **`click(x, y)`** (harness-pre-imported), then **`wait_for_load()`** once.

6. **`close()`:** no-op (**pass**) — teardown is intentionally fixture-level for Phase 17 (document single-line **`# Phase 17: teardown via browser-harness daemon lifecycle; no explicit close.`**).

7. Any stderr from **`browser-harness`** with **`returncode != 0`**: bundle into raised **`RuntimeError`** including decoded **`stderr`/`stdout`** tails (max_len 2048 chars optional — must include **`return code`** wording).

8. **`subprocess.run`** must use **`check=False`** explicitly and inspect **`returncode`**; timeouts per call ≥ **`120`** seconds default for navigate/get_text shells.
