## Objective
Detect which models support function calling and surface that as a `tool_calling: bool` flag on the `/api/providers/models/description` endpoint. The frontend chat model selector only shows tool-capable models.

## Detection strategy
1. Check provider config (`providers.json`) for `"tool_calling": true|false` flag per provider
2. If not set, attempt a lightweight probe: call the API with `tools=[minimal_tool]` and check if it succeeds
3. Cache result on the provider entry to avoid re-probing on every request

For simplicity in this phase: add a static `tool_calling` field to each provider in `providers.json`. Providers known to support it: `github_copilot`, `nvidia_nim`, `opencode_go`. Providers that may not: `ollama` (model-dependent), `omlx` (model-dependent). Default: `false` unless explicitly set.
