# Plan 03-02: EmbeddingService — sentence-transformers wrapper (no ProviderRegistry)

## Objective

Add a small, typed module that wraps **`SentenceTransformer("sentence-transformers/all-mpnet-base-v2")`** to produce **normalized 768-dim float32** vectors and serialize them with **`sqlite_vec.serialize_float32`** for `vec0` storage. This satisfies **D-03/D-04** (local model, **no** `ProviderRegistry` / `FallbackDispatcher`) and feeds **NLS-03** embedding generation used by the batch job and query encoding.

<action>
Create **new file** `lightroom_tagger/core/embedding_service.py` with:

**Module constants (exact values):**

- `TEXT_EMBED_MODEL_ID = "sentence-transformers/all-mpnet-base-v2"`
- `TEXT_EMBED_DIM = 768`

**Lazy singleton model:**

- Module-level `_model: SentenceTransformer | None = None`
- `def _get_model() -> SentenceTransformer:` — load `SentenceTransformer(TEXT_EMBED_MODEL_ID)` once into `_model`

**Core functions:**

1. `def embed_texts(texts: list[str], *, batch_size: int = 24) -> "np.ndarray":`  
   - Calls `_get_model().encode(texts, batch_size=batch_size, normalize_embeddings=True, show_progress_bar=False, convert_to_numpy=True)`  
   - Returns `np.ndarray` shape `(len(texts), 768)` dtype float32

2. `def embed_query_text(text: str) -> "np.ndarray":`  
   - `embed_texts([text], batch_size=1)[0]`

3. `def numpy_to_vec_blob(vec: "np.ndarray") -> bytes:`  
   - Assert `vec.shape == (768,)` (or squeeze 1-D)  
   - Use `sqlite_vec.serialize_float32(vec.astype("float32", copy=False).tolist())`

4. `def embed_query_to_vec_blob(text: str) -> bytes:`  
   - `numpy_to_vec_blob(embed_query_text(text))`

**Imports:** `from __future__ import annotations`, `numpy as np`, `sqlite_vec`, `SentenceTransformer`.

**No** imports from `lightroom_tagger.core.provider_registry` or `fallback`.
</action>
