## Objective

Add a strict **Pydantic** model for allowlisted catalog filter fields (D-06, D-07) and extend **`query_catalog_images`** with optional **`dominant_colors`** / **`mood_tags`** list parameters using SQLite **`json_each`** + bound **`IN (?)`** values (D-08), with unit tests. This is the data layer for the NL-search endpoint in plan **02-02**.
