# Plan 02: BestPhotosGrid posted overlay + dedupe metadata Posted chip

## Objective

Show a top-right “Posted” overlay on Identity Best Photos tiles (IDENT-04) using `ImageTile`’s `overlayBadges`, while avoiding duplicate “Posted” text in the metadata row that `ImageMetadataBadges` already renders.

## Context

- `ImageTile` wraps `overlayBadges` in `absolute right-2 top-2 flex flex-col gap-1`.
- `ImageMetadataBadges` renders `<Badge variant="success">Posted</Badge>` when `image.instagram_posted` is true.
- UI-SPEC and `05-UI-SPEC.md` typography: Badge labels use `font-semibold` (weight 600); base `Badge` may still use `font-medium` — align only if this plan touches `Badge.tsx` for posted overlay consistency.
