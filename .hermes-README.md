## After Execution

Once Hermes completes, you can use GitHub Copilot:

```bash
# In config.yaml, change:
vision_model: github_copilot/gpt-4o

# Or override per command:
python -m lightroom_tagger.cli match \
  --vision-model github_copilot/gpt-4o \
  --batch-size 5
```

Available models:
- `gpt-4o` - Fastest, recommended
- `gpt-4` - High accuracy
- `claude-3.5-sonnet` - Best descriptions
- `gemini-2.0-flash-thinking-exp-01-21` - Fast, good for batch

## Troubleshooting

**"Hermes not found"**
```bash
pip install hermes-agent
# Or: pip install --user hermes-agent
```

**"Plan not found"**
```bash
# Re-copy plan
cp docs/plans/2026-04-09-github-copilot-provider.md ~/.hermes/plans/github-copilot-integration.md
```

**"Gateway already running"**
```bash
# Check what's on port 3030
lsof -i :3030

# Kill if needed
pkill -f "copilot-api-gateway"
```

**Hermes stuck on Task 3**
- Create your GitHub token manually
- Edit `.env.copilot` with real token
- Tell Hermes: "Token is configured, please continue"

## Rollback

If something goes wrong:

```bash
cd /Users/ccanales/projects/lightroom-tagger

# Restore backup
cp lightroom_tagger/core/providers.json.backup lightroom_tagger/core/providers.json

# Reset git (if committed)
git reset --hard HEAD~1

# Stop gateway
pkill -f "copilot-api-gateway"
```

## Advanced: Resume After Pause

If Hermes stops or you disconnect:

```bash
# List recent sessions
ls -ltr ~/.hermes/sessions/ | tail -5

# Resume last session
hermes --resume <session-id>
```
